## Added

- TypeScript runtime with isolated sandbox execution
- E2B, Docker, Vercel, and Cloudflare sandbox providers
- Multi-model support (Anthropic, OpenAI, Google, OpenRouter)
- Streaming SSE API with Hono server
- TypeScript SDK (`@sandcaster/sdk`)
- Ink-based CLI with rich TUI
- 6 built-in starters: `general-assistant`, `research-brief`, `document-analyst`, `support-triage`, `api-extractor`, `security-audit`
- Composite multi-sandbox orchestration
- Speculative branching
- Skills system (`SKILL.md`)
- `sandcaster.json` configuration
- Slack bot integration
